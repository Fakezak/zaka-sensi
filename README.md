# ZAKA SENSITIVITY BOT 🎮

## Features
- 📱 **Phone-based sensitivity generation** (iPhone 15, S23, ROG 6, Redmi 12, Poco F5)
- 🔫 **13+ Weapons** including UMP, XM8, M1887, SCAR, AK47, M4A1, MP40, SKS
- 💬 **AI Chat Assistant** for instant help
- ⚡ **ASHELL Commands** (upscale 1.9, fps 90, smooth 2.0, etc.)
- 🎥 **Tricksensi-style video player** with auto-play
- 🌙 **Dark theme** with cyber animations

## Installation

1. Clone the repository
2. Run `npm install`
3. Start server: `npm start`
4. Open `http://localhost:3000`

## Usage

1. **Select your phone** from the top menu
2. **Click GENERATE** for personalized sensitivity
3. **Ask Zaka AI** about weapons or commands
4. **Try ASHELL commands** for performance boost
